# 4dx by Raspberry Pi

This is a project for making 4dx theater using Raspberry Pi.

# Download
macOS:

`curl -O https://github.com/yuji1997/4dx`

or

`git clone https://github.com/yuji1997/4dx`

Linux:

`wget https://github.com/yuji1997/4dx`
